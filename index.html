When creating a new HTML file you need the following elements:

    <!DOCTYPE html>

    <html>

    <head>

    <title> Here the title of the page will show up in the tab </title>

    </head>

    <body>

    
The majority of the webpage content goes here
 

    </body>

    </html>

A Table Going Over The Basic Elements of A Table
Item Description Example
<table> This is the main table element <table> and </table>
<caption> This is a caption text that can be placed to describe the table <caption> and </caption>
<thead> This declars the 1st row on the table, but an <tr> is needed to fill the table row <thead> and </thead>
<tr> This element fills the row and contains <th> for the heading (or 1st cell), <td> for the table data (2nd cell) <thead> and </thead>
<td> This contains the table data <td> and </td>
Here lies the <tfoot> element, which requires a <tr> and defined colspan (here it is 3) and a <td> to actually fill with data
Remember a table row <tr> must always have a <td>!!!!
